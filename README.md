# webResume
 WELCOME

 Usage
--------
 
Start the development server with this command:
 
```
npm start
```

 Setup
 Run the following command from the root directory of the project

 ```yarn ```

 Alternatively
 --------------

 ```npm install```

 Compile
---------
 ```webpack```

 Generate a minified build file for deployment
------------------------------------------------

 ```webpack --config webpack.config.prod.js -p```
